# Prompt2Print - Worksheet Platform

An AI-powered tool that turns a teacher's plain-language prompt into a
polished, visually engaging LaTeX worksheet — generated, compiled, and
previewed in one place.

## Status
🚧 In active development. See the roadmap below.

## Roadmap
1. Compiler: LaTeX → PDF, safely, in Docker
2. FastAPI endpoint wrapping the compiler
3. Claude integration: prompt → LaTeX
4. Frontend: prompt box + PDF preview
5. Auto-fix loop on compile errors
6. Conversational editing
7. Accounts & saved worksheets

## Tech stack
Python · FastAPI · Claude API · TeX Live · Docker · React
